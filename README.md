### Sortong algorithms comparison

|              | Quicksort     | Selection sort                     |
| ------------ | ------------- | ---------------------------------- |
| Worst-case   | $O(n^2)$      | $O(n^2)$ comparisons, $O(n)$ swaps |
| Best-case    | $O(n\log{n})$ | $O(n^2)$ comparisons, $O(1)$ swap  |
| Average-case | $O(n\log{n})$ | $O(n^2)$ comparisons, $O(n)$ swaps |
